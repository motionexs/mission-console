## Purpose

This workflow helps me think through how to return an athlete to jumping after pain or injury.

It does not replace a physio, doctor, or qualified medical professional.

Its purpose is to support safer coaching decisions and help me avoid rushing the athlete back too quickly.

## When To Use This

Use this workflow when:

- an athlete had pain during jumping
- an athlete is returning after injury
- jump volume was reduced because of pain
- the athlete is unsure or lacks confidence
- the athlete tolerates basic training but is not ready for full jumping
- I need a gradual return-to-jumping plan

## Step-by-Step Process

### Step 1: Check if jumping is appropriate

Before adding jump work, ask:

- Is pain present during basic movement?
- Is pain sharp or worsening?
- Does pain change the athlete’s movement?
- Is swelling, instability, or unusual weakness present?
- Has the athlete been assessed if needed?

If there are concerning signs, do not progress jumping. Refer or seek medical input.

### Step 2: Start with low-risk preparation

Begin with lower-risk options such as:

- mobility
- isometrics if appropriate
- basic strength
- balance and control
- low-impact coordination drills
- walking drills
- low-intensity rhythm work

The goal is to prepare the body before adding impact.

### Step 3: Add low-level impact

Progress to simple low-level impact if tolerated.

Examples:

- pogos
- skipping
- low-level hops
- ankling drills
- low amplitude jump contacts
- rhythm runs without takeoff

Keep volume low.

### Step 4: Add controlled jump-specific drills

If low-level impact is tolerated, progress to more specific work.

Examples:

- short approach rhythm runs
- low-intensity takeoff drills
- pop-ups
- curved approach rhythm
- controlled low bar work if appropriate

The goal is technical quality, not maximal effort.

### Step 5: Progress toward full jumping

Progress only if the athlete tolerates previous steps.

Possible progression:

1. rhythm runs
2. low-intensity takeoff drills
3. short approach jumps
4. partial approach jumps
5. full approach jumps
6. competition-style jumps

Do not jump stages too quickly.

### Step 6: Monitor the pain response

Check:

- pain during session
- pain after session
- pain later that day
- pain the next morning
- movement quality
- confidence
- soreness
- fatigue

The next-day response is important.

### Step 7: Adjust the plan

If symptoms increase, reduce load.

If symptoms stay stable and movement quality is good, progress slowly.

If symptoms worsen or become concerning, stop high-risk work and refer if needed.

## Decision Rules

- If pain is present during basic movement, then do not add jumping yet.
- If pain is sharp, worsening, or changes movement, then stop high-risk work and refer when appropriate.
- If low-level impact is tolerated, then progress gradually.
- If pain increases after the session or the next morning, then reduce the next jumping load.
- If movement quality is poor, then reduce intensity or return to an earlier stage.
- If confidence is low, then use easier progressions before full jumping.
- If the athlete tolerates several sessions well, then progress volume or intensity slowly, not both at once.

## Output

This workflow should produce:

- current return-to-jumping stage
- safe training option
- progression decision
- monitoring points
- referral flag if needed
- next session recommendation

## Mistakes To Avoid

- Returning based only on time.
- Going straight back to full jumps.
- Ignoring pain after the session.
- Ignoring next-morning pain.
- Increasing volume and intensity together.
- Skipping low-level impact preparation.
- Treating confidence as unimportant.
- Trying to act like a physio when referral is needed.
