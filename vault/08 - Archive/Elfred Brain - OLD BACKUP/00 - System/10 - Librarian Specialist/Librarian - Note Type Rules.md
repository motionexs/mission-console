# Librarian - Note Type Rules

The Librarian must classify every piece of information into one of these note types.

## 1. Concept Note

Use for general knowledge.

Examples:

- Training Load
- Fatigue
- Relative Strength
- Tendon Adaptation
- Rate of Force Development

Folder:

04 - Knowledge Base/

Naming:

Concept - Topic Name

## 2. Source Note

Use for information from a paper, book, lecture, video, course, or PDF.

Folder:

05 - Evidence Library/

Naming:

Source - Author Year - Topic

or

Source - Lecture - Topic

## 3. Workflow Note

Use for repeatable coaching, planning, rehab, or decision-making processes.

Examples:

- Build a Training Week
- Modify Training Based on Fatigue
- Return to Jumping After Pain

Folder:

03 - Workflows/

Naming:

Workflow - Process Name

## 4. Field Insight Note

Use for practical lessons from coaching, training, athletes, or real-world experience.

Folder:

04 - Knowledge Base/Field Insights/

Naming:

Field Insight - Lesson Name

Required structure:

## Situation
## What Theory Suggested
## What Happened In Practice
## Lesson Learned
## Possible Decision Rule
## Context Notes
## Links To Update
## Confidence Level
## Follow-Up


## 5. Athlete Note

Use for individual athletes, testing results, monitoring logs, and training reviews.

Folder:

02 - Athletes/

Naming:

Athlete - Code or Name

or

YYYY-MM-DD - Test Type - Athlete Code

## 6. Project Note

Use for things I am building.

Folder:

01 - Projects/

Naming:

Project - Project Name
