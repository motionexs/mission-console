# Source Review - Aerobic Endurance Exercise Training

## What This Source Taught Me

Aerobic endurance training is not just about doing more cardio.

It is about controlling:

- mode
- frequency
- intensity
- duration
- progression
- recovery
- fatigue

The biggest idea is that endurance performance depends on multiple factors, not only VO2 max.

## Coaching Meaning

For endurance athletes, these concepts help design better training programs.

For power athletes like high jumpers, this source helps me understand how to use aerobic work carefully.

Aerobic work can support:

- general preparation
- recovery
- work capacity
- basic conditioning

But it should not interfere with:

- speed
- power
- technical quality
- jump freshness

## Key Decision Rules

- If intensity increases, duration usually needs to decrease.
- If duration increases, intensity usually needs to decrease.
- If recovery is poor, reduce frequency, intensity, or duration.
- If the athlete is a high jumper, aerobic work should support the main training goal, not replace it.
- If the session goal is recovery, the aerobic work must stay easy enough to avoid extra fatigue.
