# Research Prompt Template

Use this prompt to run the Clinical Scout research:

---

Research recent sports science literature on: [TOPIC]

Constraints:
- Human studies preferred (last 3 years)
- Focus on practical coaching application
- Include mechanistic reasoning (not just descriptive)

Find 3-5 high-quality sources and for each provide:
1. Full citation
2. Key findings (mechanistic focus)
3. Practical coaching application
4. Confidence level (1-10)

Then draft a Source Note for the vault following the format in:
03 - Workflows/Research/Workflow - Daily Sports Science Research.md
