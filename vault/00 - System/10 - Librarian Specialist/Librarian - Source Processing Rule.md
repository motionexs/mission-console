# Librarian - Source Processing Rule

## Purpose

This rule tells the Librarian how to process PDFs, lectures, textbook sections, research papers, videos, and other source material.

The Librarian must not turn every source into many notes at once.

The first job is always to create a clean Source Note.

---

## Main Rule

When I upload or paste source material, the Librarian must first create:

1. Source Note
2. Key concepts from the source
3. Suggested links
4. Decision rules
5. Questions or gaps
6. Next best concept to create

---

## Source Note Folder

All source notes go in:

05 - Evidence Library/Source Notes/

---

## Source Note Naming

Use this format:

Source - Type - Topic

Examples:

- Source - Lecture - Aerobic Endurance Exercise Training
- Source - Book - Periodization Basics
- Source - Research Paper - Tendon Adaptation in Jumpers

---

## Required Source Note Structure

Every Source Note must include:

## Source Summary
## Key Points
## Important Terms
## Useful Ideas for Coaching
## Possible Decision Rules
## Concepts Created From This Source
## Existing Concepts This Connects To
## Workflows Affected
## Questions or Gaps
## My Notes

---

## Important Rule

The Librarian must separate:

- What the source says
- What I think about it
- How it applies to coaching
- What still needs to be tested or researched

---

## Linking Rule

A Source Note should link to:

- concepts created from the source
- existing concepts already in the vault
- workflows affected by the source

---

## Do Not Overproduce Rule

After creating a Source Note, the Librarian should suggest many possible concept notes but only recommend creating one next concept at a time.

Example:

Do not create 12 concept notes at once.

Instead:

1. Create the Source Note
2. Create one important Concept Note
3. Link both ways
4. Continue slowly
---

## Source Review Rule

After a source has been processed into concept notes, create one Source Review note.

The Source Review note should answer:

- What did this source teach me?
- What does this mean for coaching?
- What decision rules did I learn?
- What questions do I still have?
- Which workflows may need updating later?

Source Review notes should be named:

Source Review - Source Topic

Example:

Source Review - Aerobic Endurance Exercise Training

The Source Review should link back to the original Source Note.
